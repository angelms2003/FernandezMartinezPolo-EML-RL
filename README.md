## Descripción
En este directorio se presenta todos los notebooks y el código creado y usado en nuestros experimentos para la Práctica 1 sobre aprendizaje por refuerzo de la asignatura Extensiones de Machine Learning del Master en Inteligencia Artificial de la Universidad de Murcia.

## Estructura
El repositorio se divide en dos directorios: k-brazos y Entornos Complejos. 

En el primero de ellos se encuentran todos los notebooks generados en nuestra experimentación con el problema de los k-bandidos. Además, se incluye un notebook "main" con enlaces a todos los notebooks y con la creación de algunas gráficas "conjuntas" que unen resultados de los otros notebooks. También se encuentra el subdirectorio src, donde se encuentra el código Python necesario para ejecutar nuestros experimentos.

El segundo de ellos sigue la misma estructura que el anterior. En este caso se proporciona todo lo experimentado en nuestra práctica con Entornos compeljos y el código necesario para reproducirlo.

También se incluye en este directorio raiz otro notebook main con enlaces a los notebooks main de ambas partes.

## Instalación y Uso
Con descargar el directorio y mantener su estructura, se deben poder ejecutar nuestros experimentos de los notebooks sin problema.

## Tecnologías Utilizadas
Para este proyecto se ha trabajo solamente con Python y Jupyter Notebooks. Se han usado librerías de python específicas como Gymnasium pero todas los requisitos son tratados en los notebooks correspondientes.
