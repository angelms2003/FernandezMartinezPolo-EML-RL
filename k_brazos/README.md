## Descripción
Este trabajo presenta un estudio comparativo sobre el rendimiento de diferentes algoritmos de aprendizaje por refuerzo aplicados al problema del bandido multibrazo.

Se han evaluado algoritmos pertenecientes a las siguientes familias:

- **$\epsilon$-greedy:** se implementó el algoritmo $\epsilon$-greedy y se implementó la opción de $\epsilon$-decaimiento.
- **Softmax** se evaluó el algoritmo Softmax.
- **UCB (Upper Confidence Bound):** se probaron las variantes UCB1 y UCB2.

El objetivo de este estudio es comparar la eficiencia y comportamiento de cada estrategia bajo distintas condiciones de recompensa, proporcionando un análisis empírico sobre su desempeño en escenarios típicos del problema de los bandidos de $k$-brazos.
